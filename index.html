【WebRTC研究（5）】RTMP与WebRTC互转方案预研——OvenMediaEngine

发表于 2020-02-20 分类于 技术 ， 音视频 ， WebRTC Facebook： 0

什么是OvenMediaEngine？

OvenMediaEngine（OME）是一个开源的超低延迟流服务器。OME通过RTMP从实时编码器（例如OBS，XSplit）接收视频，并将其通过WebRTC传输。因此，来自OME的超低延迟流可以在您的浏览器中无缝运行而无需插件。另外，OME提供了OvenPlayer，这是HTML5标准的网络播放器。

我们的目标是使您能够更轻松地构建具有超低延迟的稳定广播/流服务。

特征

  • RTMP输入
  • WebRTC / HLS / MPEG-DASH流
  • 嵌入式实时转码器（VP8，H.264，Opus，AAC）
  • 嵌入式WebRTC信令服务器（基于WebSocket）
  • ICE（交互式连接建立）
  • DTLS（数据报传输层安全性）
  • SRTP（安全实时传输协议）
  • 带有VP8，H.264的ULPFEC（前向纠错）
  • 带Opus的带内FEC（前向纠错）
  • P2P交付（预览版）
  • 高可用性
  • 聚类
    • 原点边缘结构

支持平台

我们已经在下面列出的平台上测试了OME。但是，我们认为它也可以与其他Linux软件包一起使用：

  • Docker
  • Ubuntu 18
  • CentOS 7
  • Fedora 28

雅安碧峰峡-20191214

发表于 2019-12-11 更新于 2020-01-21 分类于 旅游 Facebook： 0

出行计划

景点

  • 廊桥（夜景好一点）

  • 上里古镇

  • 碧峰峡野生动物园

住宿

  • 雅安市雨城区碧峰峡镇碧峰村3组 云海美宿
  • 电话： 18090108499

美食

  • 百丈马溜坡正宗叶羊肉
  • 贡椒鱼老店/干老四雅鱼饭店

成行游记

第一次带小孩出来旅游，挑了一个离家比较近的景点。开车全程用了不到2个小时。先到贡椒鱼老店，雅鱼168一斤，想想还是算了吧，点了江团。不得不说雅安的鱼很鲜、很嫩，值得品尝。

然后到宾馆放下行李，又去廊桥转了一圈。找停车位找了至少半个小时，没有地下停车场，全部停在路边，也侧面验证了雅安确实是一个很小的城市。夜景一般，适合附近的人晚餐后出来散步。但如果作为一个景点来说，就有点太单调了。反而小孩在一个大鞋子里面玩了半天不肯走。

晚上回去的时候是山路，没有路灯，在几乎没有车的路上速度没敢超过40。心惊胆战的到达住处，我激动的差点哭出来——莫名其妙。因为是在网上随意选了一个山上的民宿，环境一般，说实话下次我肯定会选择稍微好一点的酒店，带自助早餐、并且品种很多的那种。但是小孩却是异常兴奋，在床上又蹦又跳。

第二天早早的起床，天空下着濛濛细雨。因为离景区门口很近，我们可能是最早的一批上山游客。额外加了200块钱乘坐了猛兽观光车，让孩子近距离接触一下老虎、狮子、黑熊。因为角度原因，没法拍照，不过并没有遗憾，非常值得。

然后依次游览了羊驼、梅花鹿、猴山、丹顶鹤等，还观看了老虎、大象表演，非常充实。最后快出山门的时候，特意花了20块钱让小孩玩了一下蛇。不知道是初生牛犊不怕虎，还是天生勇敢不惧蛇。绕在脖子上玩了半天。我一直在担心蛇失控会用力勒住脖子，但表面上还得故作轻松——哎呀。

值得一提的是，整整走了将近3个小时，小家伙竟然没喊累，看来下次带他去爬青城山应该没问题了。

最后去吃了群里大力推荐的百丈马溜坡正宗叶羊肉，结果发现我对这种美食好像不太感冒，评价一般，反正不会带朋友去吃。

至此，打道回府。
